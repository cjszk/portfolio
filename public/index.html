Projects

Flow Tools

Flow tools is my response to a lifestyle of demanded focus and productivity. Designed specifically for busy computer users, Flow tools is an application that aims to improve the productivity of computer users by providing a simple and quick way to organize task-management chronologically with notifications for overdue, current and upcoming tasks. Flow tools also provides convenient integrated widgets to maximize productivity.

Why did I make this? I always have felt the pressure of wanting to learn so much, with so little time. Because of my unending thirst for knowledge, I've naturally been fascinated with how to accelerate learning. Maximizing productivity is one key aspect to accelerating learning- this is what I address with Flow Tools- to create an environment that maximizes productivity for the regular computer user.

Tech Used: HTML/SASS/JS: React, Node/Express, PostreSQL (w/ knex.js)

Macronator

Macronator is a fitness application designed to help individuals with their body composition goals. The user inputs their weight measurements and calories consumed over certain dates. The application runs a mathematical formula with all inputted data to calculate the user's total daily energy expenditure and gives them advice to help them reach their goals. These updated results are the work of multiple data inputted over time by the user.

Why did I make this? Many fitness applications calculate a user's 'estimated' energy expenditure, but almost none go as far as to calculate it accurately based off individual user feedback, despite the fact that we have the scientific understanding of how to calculate this. Macronator addresses this issue by using constantly evolving calculations with each user data input.

Tech Used: HTML/SASS/JS: JQuery, Node/Express, MongoDB
